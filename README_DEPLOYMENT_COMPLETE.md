### Step 2: Railway Backend (10 min)
```
1. Go to https://railway.app
2. Sign in with GitHub
3. Create New Project from GitHub
4. Select: IshaanIshu13/cakesman-bakery
5. Set Root Directory: backend
6. Add environment variables:
   - PORT=5001
   - MONGO_URI=mongodb://localhost:27017/cakesman
   - JWT_SECRET=cakesman_secret_key_2025
   - NODE_ENV=production
7. Deploy!
```
**You'll get a URL like**: `https://cakesman-bakery-backend-xxx.railway.app`

### Step 3: MongoDB Atlas Database (5 min)
```
1. Go to https://mongodb.com/cloud/atlas
2. Sign up with email/GitHub
3. Create Free Cluster
4. Create Database User
5. Whitelist All IPs (0.0.0.0/0)
6. Get Connection String
```
**Format**: `mongodb+srv://admin:password@cluster.mongodb.net/cakesman`

### Step 4: Update Railway with MongoDB (2 min)
```
1. In Railway dashboard
2. Update MONGO_URI to MongoDB Atlas connection string
3. Save
4. Auto-redeploys!
```
